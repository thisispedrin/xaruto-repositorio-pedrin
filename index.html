Exemplo de conteúdo

Este é um exemplo simples de página. O cabeçalho está fixo no topo, a imagem substitui o "X" de "Xaruto" — apenas salve sua imagem do charuto no mesmo diretório com o nome cigarrilha.png (ou troque no <img>).

O efeito de fumaça é feito em CSS (sem GIF). Se preferir que a fumaça seja exatamente a partir da ponta do seu arquivo de imagem, me envie o arquivo e eu ajusto o posicionamento.

Como usar

  1. Baixe este arquivo (salve como index.html).
  2. Salve a imagem do charuto no mesmo diretório com o nome cigarrilha.png (ou altere o atributo src).
  3. Abra o index.html no navegador.
